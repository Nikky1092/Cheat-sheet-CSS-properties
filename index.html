Padding

Margin

Padding: is the space between the content of a box and the its border.

Margin: is the space outside of the box, surrounding its border. It specifies the size of this space.

CSS Padding and Margin definition values

.example1{
padding:80px;
margin: 20px;
}

One value is used when the same quantity is applied to all four sides of the content.

.example2{
padding: 10px 15px ;
margin: 50px 5px;
}

Two values are used to define both vertical (top and bottom) spacing and horizontal (left and right) spacing for padding and margin.

.example3{
padding:80px 120px 100px;
margin: 2px 10px 5px;
}

We use 3 values when left and right spacing values (horizontal spacing)are equal and you want to define the spacing for all 4 sides of the element.

.example4{
padding:80px 100px 120px 140px;
margin: 3px 10px 1px 2px;
}

We use 4 values when we want to define a different spacing for all 4 sides.

Purpose & Common use

Padding ensures that the content of the object doesn't touch the edges that covers it (i.e. its borders)

Margin ensures that the object is not being interfered or affected by the other objects that surround it.

Behavior next to other elements: Collapsation

Element's Padding & horizontal Margins (left and right margins) DO NOT COLLAPSE, but VERTICAL MARGINS-top and bottom margins- DO COLLAPSE.

It is a phenomenon which occurs when the top or bottom margin of one element comes into contact with the top or bottom margin of another element. Only vertical margins collapse, horizontal margins do not.

In collapsing of two margins, the distance between the two boxes will use whichever margin value is greater. Margin collapsing also happens when an element is nested inside another element. Irrespective of contact of elements, even an element inside of another element will margin collapse with its parent if the two margins come into contact. Prevention of margin collapsing can be done by applying padding or a border to the either of the element.

Effect on element's size

Margin increases the size of the object while padding has no impact on the object's size. Padding doesn't influence the width nor height of the element.
