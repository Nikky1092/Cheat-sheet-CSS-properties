Definition

Padding

Is the space between the content of a box and the border. It ensures that the content of the object doesn't touch the edges that covers it (i.e. its borders)

Margin

Is the space outside of the box, surrounding its border. It specifies the size of this space and ensures that the object is not being interfered or affected by the other objects that surround it.

Values

.example1{
padding:80px;
margin: 20px;
}

One value is used to apply the same spacing to all four sides of the content.

.example2{
padding: 10px 15px ;
margin: 50px 5px;
}

When using 2 values, the first value is used to define vertical (top and bottom) spacing and the second value is used to define horizontal (left and right) spacing for padding and margin properties.

.example3{
padding: 80px 120px 100px;
margin: 2px 10px 5px;
}

When using 3 values, the first value is used to define the top spacing, the second value is used to define the same spacing for left and right sides of the content and the third value is used to define the bottom spacing.

.example4{
padding: 80px 100px 120px 140px;
margin: 3px 10px 1px 2px;
}

We use 4 values when we want to define a different spacing for all 4 sides.

Collapsation

Padding & Horizontal Margins (left and right margins) DO NOT COLLAPSE, but VERTICAL MARGINS-top and bottom margins- DO COLLAPSE.

It is a phenomenon which occurs when the top or bottom margin of one element comes into contact with the top or bottom margin of another element. Only vertical margins collapse, horizontal margins do not.

In collapsing of two margins, the distance between the two boxes will use whichever margin value is greater. Margin collapsing also happens when an element is nested inside another element. Irrespective of contact of elements, even an element inside of another element will margin collapse with its parent if the two margins come into contact. Prevention of margin collapsing can be done by applying padding or a border to the either of the element.

INfluence on Size

Margin increases the size of the object.

Padding, on the other hand, doesn't influence the width nor height of the element. It has no impact on the object's size.
